class jlab_archiver_client.point.Point(query, url=None)[source]

Bases: object

A class for running calls to myquery’s point endpoint.

This endpoint returns a single channel event. The user supplies the channel name and a timestamp, and myquery return the closest event before (inclusive) the timestamp.

Options exists to look into the future and exclude the given timestamp from search space. That is helpful in the scenario that you want to know the next event before/after an event you already know of.
